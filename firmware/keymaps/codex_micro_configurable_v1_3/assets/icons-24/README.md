# 24 px layer icons

Four monochrome 24 x 24 SVG layer icons for the DOIO KB16 OLED:

- `codex.svg`: terminal prompt
- `number.svg`: calculator
- `navigation.svg`: four-way navigation
- `system.svg`: settings gear

All four icons use the Tabler Icons outline family with a 24 x 24 viewBox,
2 px stroke, round caps and round joins. The source icons are MIT licensed:
https://github.com/tabler/tabler-icons

The files are design assets only. They have not yet replaced the compiled OLED
bitmaps in `oled_icons.c`.
